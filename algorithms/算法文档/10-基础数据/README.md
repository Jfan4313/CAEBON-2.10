## 季节系数

### 季节分类

| 季节 | 月份 | 负荷系数 | 空调负荷乘数 | 照明负荷乘数 | 光伏效率乘数 |
|-----|------|---------|-------------|-------------|-------------|
| 春季 | 3-5月 | 0.9 | 1.1 | 1.05 | 1.0 |
| 夏季 | 6-8月 | 1.2 | 1.4 | 1.1 | 0.95 |
| 秋季 | 9-11月 | 1.0 | 1.1 | 0.95 | 1.0 |
| 冬季 | 12-2月 | 0.8 | 0.85 | 0.9 | 0.9 |

### 说明

- **负荷系数**：整体用电负荷的季节性调整系数
- **空调负荷乘数**：夏季空调负荷最高
- **照明负荷乘数**：冬季照明需求较高
- **光伏效率乘数**：夏季温度高，光伏效率略降

## 使用方法

### 获取城市日照数据

```typescript
import { getCitySunshineData } from '../ai-platform-algorithms';

// 获取上海日照数据
const shanghaiSunshine = getCitySunshineData('上海');
console.log(`上海年日照时数: ${shanghaiSunshine} 小时`);
```

### 生成默认电价曲线

```typescript
import { generateDefaultPriceCurve } from '../ai-platform-algorithms';

// 生成默认电价曲线
const priceCurve = generateDefaultPriceCurve({
  peakPrice: 1.2,
  valleyPrice: 0.35,
  flatPrice: 0.6
});

console.log(`24小时电价: ${priceCurve.hourlyPrices}`);
```

### 生成默认人流量模式

```typescript
import { generateDefaultTrafficPattern } from '../ai-platform-algorithms';

// 生成默认人流量模式
const trafficPattern = generateDefaultTrafficPattern();

console.log(`工作日模式: ${trafficPattern.workdayPattern}`);
console.log(`周末模式: ${trafficPattern.weekendPattern}`);
```

## 数据来源

- **城市日照数据**：中国气象局、NASA可再生能源数据库
- **建筑热导系数**：建筑节能设计标准GB 50189-2015
- **季节系数**：基于历史气象数据和用电负荷统计
- **光照等级**：国际标准ISO 9050
